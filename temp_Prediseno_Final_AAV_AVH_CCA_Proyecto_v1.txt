Prediseño Final del ProyectoTesis Doctoral / Manuscrito MaestroAlucinaciones Auditivas Verbales (AAV/AVH) como ejeMarco Integrado: Medicina Psiquiátrica • Neurología • Acústica/Física • Telecom/DSP • Forense Defensivo • Farmacología • Estadística/Probabilidad • Ética1. Propósito del prediseñoEste documento fija el diseño operativo final del proyecto: qué debe contener el manuscrito, qué salidas (deliverables) se requieren, cómo se extraerán y contabilizarán todas las fuentes internas (de cualquier tipo) y cómo se controlará calidad, deduplicación, trazabilidad y ética. Está alineado con los desarrollos establecidos: CCA+FPI, escenarios A–D, evidencia E0–E4, stop-rules, separación estructura vs semántica, enfoque defensivo y anti-iatrogénico.2. Alcance y no-alcance (reglas duras)2.1 AlcanceTema central: AAV/AVH (“oír voces”) desde medicina psiquiátrica con integración técnico-física y forense.Marco integrador: CCA + FPI (Fuente–Percepción–Interpretación); escenarios A–D; evidencia E0–E4.Método reproducible: instrumentación, metrología, diseños ABAB, análisis estadístico/bayesiano y reglas de parada.Inventario total de fuentes internas: papers, libros, guías, estándares (ISO/IEC/ITU/IETF), NIST, documentación técnica, datasets, enlaces web, etc.2.2 No-alcanceNo se incluye instrucción ofensiva/operacionalización de intrusión, vigilancia o ataque. Enfoque estrictamente defensivo.No se presentan diagnósticos clínicos individuales ni recomendaciones clínicas personalizadas.No se infiere atribución (culpables/actores) sin evidencia E4 y estándares probatorios apropiados.3. Arquitectura del manuscrito (estructura final)El manuscrito final se organiza en módulos. Cada módulo debe contener: definición operacional, hipótesis rivales, variables observables, predicciones diferenciales, pruebas/controles, criterios de refutación, sesgos y consideraciones éticas.M0 Preliminares: glosario, notación, convenciones, mapa capítulo→pregunta→evidencia→refutación.M1 Núcleo psiquiátrico: fenomenología AAV, evaluación estructurada, riesgo, límites DSM/ICD.M2 Diferencial médico: ORL/audiología, sueño, metabólico, inflamatorio/autoinmune, tóxicos/sustancias.M3 Neurología: vías auditivas, TLE/auras, lesiones, EEG/evocados, modelos funcionales.M4 Neurocognición y lenguaje: predictive processing, priors, pareidolia auditiva/algorítmica, estructura vs significado.M5 Psicoacústica y física: SNR/masking, modulación, acústica de recintos/estructural, no linealidades, metrología.M6 Telecom/DSP: códecs, jitter, PLC, VAD/DTX/CNG, AEC/NS/AGC, taxonomía de artefactos tipo habla, pruebas ON/OFF.M7 Forense defensivo: cadena de custodia, auditoría reproducible, baseline, límites de inferencia; stalkerware como riesgo (no acusación).M8 Modelo integrador CCA+FPI: escenarios A–D, matriz E0–E4, reglas de reporte neutral.M9 Hipótesis y predicciones: H/T/P/R/F/F-fármaco; criterios de refutación; condiciones de invalidez.M10 Metodología y estadística: diseño experimental, paneles ciegos, confiabilidad (kappa/ICC), frecuentista + bayesiano + decisión bajo incertidumbre.M11 Resultados/discusión: interpretación por E0–E4, sesgos, implicaciones clínicas, iatrogenia y reducción de daño.M12 Ética/DDHH/legalidad: privacidad, dignidad, gobernanza del dato, comunicación responsable.M13 Sistema editorial: prompts/workflows, control de versiones, reconciliación de contradicciones, trazabilidad del corpus.Anexos mega: compendio de chats, compendio de documentos, tablas maestras, protocolos, scripts, bibliografía maestra deduplicada.4. Inventario total de fuentes: especificación4.1 Definición operativa de “fuente”Se considerará fuente cualquier elemento citacional o justificativo que aparezca dentro de los documentos del proyecto: entrada bibliográfica formal, DOI, ISBN, URL, RFC, estándar ISO/IEC/ITU/ANSI, documento NIST, guía clínica, dataset, software/herramienta, whitepaper, documento institucional, y enlaces web (incluyendo divulgación).4.2 Dos conteos obligatorios (ambos)Conteo A — Fuentes únicas: deduplicadas por DOI/ISBN/RFC/ID estándar o URL canónica + normalización de títulos.Conteo B — Apariciones totales: cuántas veces aparece/cita cada fuente a través del corpus (frecuencia).4.3 Metadatos mínimos por fuente (etiquetado)Tipo: paper / revisión / libro / guía clínica / estándar / RFC / NIST / dataset / software / web-divulgación / web-enciclopedia / otro.Dominio: psiquiatría / neurología / ORL-audiología / sueño / farmacología / psicoacústica / acústica / telecom-DSP / forense / estadística / ética.Calidad-auditabilidad: alta (primaria/estándar), media (secundaria/whitepaper), baja (enciclopedia/blog no revisado).Uso en el manuscrito: núcleo (evidencia), contexto, definición, método, métrica, ejemplo, contraejemplo.Estado: verificada / pendiente de verificación / descartada (duplicado espurio o mala atribución).5. Pipeline técnico para extraer y contabilizar TODAS las fuentes internas5.1 Insumos requeridos (acceso estable)Carpeta única en Drive: “AAV_AVH_MASTER_CORPUS” con todos los documentos fuente (Docs/Word/PDF/txt).Permiso: lectura para el conector.Regla: no mover IDs; añadir nuevas versiones como archivos nuevos o subcarpetas por versión.5.2 Extracción brutaBibliografías formales: secciones ‘Referencias’, ‘Bibliografía’, ‘Obras citadas’.Identificadores: DOI, ISBN, RFC, ISO/IEC/ITU/ANSI, NIST SP, PubMed IDs, arXiv/bioRxiv IDs.URLs: capturar y normalizar (quitar tracking).Ruido de extracción: detectar código/logs; clasificar como NO-fuente salvo documentación oficial de herramienta.5.3 Normalización y deduplicaciónNormalizar: título, autores, año, venue, DOI/ISBN, URL canónica.Deduplicar por: DOI/ISBN/RFC/estándar → fuzzy título → URL canónica.Resolver colisiones: preferir primaria/estándar; mantener equivalencias (preprint→paper final).5.4 SalidasTabla 1: Fuentes únicas (metadatos completos).Tabla 2: Apariciones totales (por documento y global).Tabla 3: Cobertura por capítulo (fuentes núcleo por submódulo).Lista: Pendientes de verificación (citas incompletas).6. QA y gobernanza6.1 QA bibliográficoEl núcleo argumental debe apoyarse mayormente en alta auditabilidad (primarias/estándares/guías).Trazabilidad: cada afirmación fuerte enlaza a al menos una fuente verificable.Marcas: [CITA NECESARIA], [FUENTE DÉBIL], [CONFLICTO], [REFUTACIÓN POSIBLE].6.2 QA de inferencia y éticaNo-atribución: separar ‘fenómeno’ de ‘causa’; prohibir saltos narrativos sin E4.Stop-rules para evitar escalada interpretativa cuando el dato no alcanza.Privacidad: minimización de datos sensibles; anonimización; gobernanza del acceso a anexos.7. Entregables finales7.1 ManuscritoTesis principal + anexos, con estructura modular (M0–M13).Tabla maestra: hipótesis→predicción→prueba→refutación→sesgos.Protocolos reproducibles (calibración, multicanal, ABAB, panel ciego).7.2 Bibliografía y fuentes (lo solicitado)Inventario total de fuentes internas (únicas + apariciones) en: CSV + APA7 + BibTeX (Zotero).Reporte de deduplicación (qué se fusionó y por qué).Reporte de cobertura (temas sub-citados).7.3 Material operativoCompendio de chats indexado; compendio documental con trazabilidad.Checklists: clínico-diferencial, acústico, telecom/DSP, forense defensivo, ética/gobernanza.Registro de versiones y reglas editoriales finales.8. Decisiones a fijar (evitar ambigüedad)Completar antes de cerrar versión doctoral definitiva:Formato de bibliografía principal: ( ) APA7  ( ) IEEE  ( ) Vancouver  ( ) Mixto (definir).Regla para web/divulgación: ( ) solo contexto  ( ) permitido con validación  ( ) apéndice web.Tratamiento preprint→paper: ( ) preferir final  ( ) mantener ambos con equivalencia.Política de anexos: ( ) público  ( ) privado  ( ) híbrido (definir qué va dónde).Nivel de granularidad del índice: ( ) actual  ( ) +1 nivel  ( ) +2 niveles.9. Plan de ejecución (compacto)Fase 1: Consolidación del corpus (carpeta única + versionado).Fase 2: Extracción bruta de fuentes + normalización + deduplicación.Fase 3: Etiquetado (tipo/dominio/calidad/uso) + conteo único y total.Fase 4: Mapa de cobertura + gap analysis (temas sin sustento fuerte).Fase 5: Integración en manuscrito + QA ético + QA estadístico.Fase 6: Congelación de versión (release) y anexos finales.10. Checklist de clases de fuentes esperadas (control)Guías/consensos: DSM, ICD, APA/NICE/AASM, etc.Neurología: epilepsia temporal, lesiones, EEG/evocados, revisiones top.ORL/audiología: hipoacusia, tinnitus, alucinación musical, baterías clínicas.Farmacología: psicofármacos, sedación como confusor, toxicología/interacciones.Psicoacústica/acústica: masking, modulación, recintos, vibración, metrología.Telecom/DSP: RFC/ITU, Opus, WebRTC, AEC/NS/AGC/VAD/PLC, métricas QoE.Forense: NIST, cadena de custodia, auditoría reproducible, hardening defensivo.Estadística/probabilidad: Bayes, causalidad, confiabilidad, series temporales.Ética/DDHH: Helsinki/Belmont, gobernanza de datos, privacidad.
