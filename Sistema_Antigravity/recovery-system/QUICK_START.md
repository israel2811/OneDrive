Copia y pega esto:
```
Lee C:\Users\Lenovo\OneDrive\Sistema_Antigravity\recovery-system\STATUS_TRACKER.md y continúa la tarea más reciente.

### 2. SI UNA CONVERSACIÓN SE TRABA

```

### 3. SI NECESITAS DESBLOQUEAR TODO
Copia y pega esto:
```

```

Copia y pega esto:
```
Ejecuta /persist - guarda el estado actual para continuar después.
```

## 🔧 WORKFLOWS DISPONIBLES

| Comando | Qué Hace |
|---------|----------|
| `/recovery` | Recupera una conversación trabada |
| `/unblock` | Desbloquea todas las tareas |
| `/continue` | Continúa donde se quedó |
| `/persist` | Guarda estado antes de terminar |

## 🚀 PRINCIPIOS DEL SISTEMA

1. **NUNCA PIERDAS CONTEXTO** - Siempre lee STATUS_TRACKER.md
2. **NUNCA TE DETENGAS** - Si algo falla, hay alternativas
3. **SIEMPRE DOCUMENTA** - Actualiza el estado después de cada acción
4. **SIEMPRE HAY UN CAMINO** - Ninguna tarea es imposible

---

## 💡 TIPS

- **Sé específico**: Mientras más claro el objetivo, mejor la ejecución
- **Usa los prompts**: Están diseñados para forzar la continuación
- **Guarda frecuentemente**: Ejecuta /persist cada 30 minutos
- **No temas reiniciar**: El prompt "Reset completo" funciona muy bien
