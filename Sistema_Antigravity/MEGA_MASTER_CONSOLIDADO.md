<a name="troubleshooting"></a>
# 🛠️ TROUBLESHOOTING Y SOLUCIÓN DE ERRORES

## Errores Comunes y Soluciones:

### "Agent terminated due to error"
**Causa**: Variables de entorno faltantes, timeouts de MCP
**Solución**:
1. Reiniciar conversación
2. Usar prompt de recuperación nuclear
3. Dividir tarea en pasos más pequeños

### Chat se queda en silencio
**Solución**:
```
Responde ahora. Continúa la tarea anterior sin esperar confirmación.
```

### Herramienta falla repetidamente
**Solución**:
1. Intentar con parámetros diferentes
2. Usar herramienta alternativa
3. Descomponer en pasos manuales

### Pérdida de contexto
**Solución**:
```
Lee estos archivos en orden:
1. OneDrive\Sistema_Antigravity\recovery-system\MASTER_RECOVERY.md
2. OneDrive\Sistema_Antigravity\recovery-system\STATUS_TRACKER.md
Luego continúa la tarea pendiente más importante.
```
