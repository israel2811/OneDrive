# Chapter 34: Publication Strategy and Dissemination

## 34.1 Introduction: From Thesis to Impact

Research that remains unread has no impact. This chapter outlines the publication strategy for CCA research, from minimum viable product to maximum potential, with attention to audience, venue, and communication ethics.

## 34.2 Go/No-Go Decision Points

### 34.2.1 Minimum Defendable Product

If empirical results are weak or null:

| Deliverable | Content | Value |
|-------------|---------|-------|
| Theoretical framework | CCA + FPI + E0-E4 | Conceptual contribution |
| Methodology | P0-P4 Protocol | Methodological contribution |
| Instruments | CCA-CIP, SPIA-SS, RCTT | Clinical tools |
| Critical review | "Sin estímulo externo" critique | Epistemological contribution |

**Thesis defense**: Viable as theoretical/methodological contribution even without strong empirical confirmation.

### 34.2.2 Maximum Potential

If empirical results are robust (E3-E4):

| Deliverable | Content | Impact |
|-------------|---------|--------|
| All minimum + | Validated instruments | Clinical application |
| Empirical papers | Case series, N-of-1 | Evidence base |
| Technical standards | APF implementation | Engineering contribution |
| Policy recommendations | Assessment protocols | Healthcare policy |

**Career trajectory**: Multiple publications, potential for funded research program.

### 34.2.3 Decision Criteria

| Finding | Interpretation | Action |
|---------|----------------|--------|
| No correlation acoustics-SPIA | Model not supported | Publish null findings + methodology |
| Weak correlation (r < 0.3) | Small effect | Report cautiously; suggest replication |
| Moderate correlation (r 0.3-0.5) | Promising | Publish with appropriate caveats |
| Strong correlation (r > 0.5) | Model supported | Comprehensive publication program |

## 34.6 Communication Ethics

### 34.6.1 Avoiding Harm

| Risk | Mitigation |
|------|------------|
| Reinforcing conspiracy theories | Explicit disclaimers; calibrated language |
| Stigmatizing voice-hearers | Non-pathologizing language |
| Undermining legitimate psychiatry | Affirm role of clinical assessment |
| Enabling abuse detection avoidance | Don't provide evasion techniques |
| Creating false hope | Honest about evidence levels |

### 34.6.2 Responsible Communication

- State evidence levels explicitly (E0-E4)
- Distinguish hypothesis from conclusion
- Report null findings
- Acknowledge limitations prominently
- Avoid sensational headlines

### 34.6.3 Media Guidelines

If media coverage occurs:
- Prepare key messages
- Correct misquotations promptly
- Direct to peer-reviewed sources
- Avoid speculation beyond data
- Emphasize preliminary nature of findings

## 34.7 Intellectual Property

### 34.7.1 Open Access Philosophy

Commitment to open science:

- Final versions via green open access
- Code and data open where possible
- Instruments freely available for clinical use

### 34.7.2 Attribution

## 34.10 Conclusion: Research with Purpose

Publication is not the end goal—impact is. By targeting appropriate audiences, venues, and timing, CCA research can contribute to:
- Better differential diagnosis
- Reduced iatrogenia
- Improved patient experience
- Technology accountability

The measure of success is not papers published, but lives improved.
